# Dogs Data Analysis 🐕

A comprehensive data analysis project focused on dog breeds, their characteristics, and statistical insights.

## Project Overview

This project analyzes various dog breeds and their physical characteristics, including weight, size, and other relevant data points. The analysis helps identify patterns and interesting facts about different dog breeds.

## Key Findings

- **Heaviest Breed**: St. Bernard with an average weight of 75.0 kg
